### 拡張案
- ランダムマッチング1VS1
- 接続人数の動的変更

### 開発
1. npm run build
1. node src/server.js

### デプロイ
1. git add .
1. git commit -m "first commit"
1. heroku login
1. heroku keys:add
1. heroku create
1. git push heroku master
1. heroku rename "<好きな名前（既に使われている場合、使用不可）>"
